# Crop Yield Analysis (Week 3 Assignment 1)

## Overview

This project analyzes agricultural data to understand the factors affecting crop yield.
The analysis includes data visualization, correlation study, and feature engineering.

---

## Key Tasks Performed

* Data exploration and summary statistics
* Visualization:

  * Histogram (Yield distribution)
  * Scatter plots (Rainfall & Fertilizer vs Yield)
  * Count plots (Soil types)
* Correlation analysis with heatmap
* Group-based analysis (Crop type & Soil type)
* Feature encoding (One-Hot Encoding)
* Feature selection

---

## Key Insights

* Rainfall and irrigation have the strongest impact on crop yield
* Fertilizer also shows a positive correlation
* Rice has the highest average yield
* Loamy soil performs best among all the soil types
